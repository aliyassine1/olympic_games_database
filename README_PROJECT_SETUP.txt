1. unzip the project.

2. open phpmyadmin, create a new database and name it "olympic", 
then import the "olympic.sql" file.

3. open the project in Netbeans.

4. inside Netbeans, add the .JAR file and choose the "connector supported by the zip 
file with this project, in order to be able to connect to phpmyadmin. (SQL Connector).
